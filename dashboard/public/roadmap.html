Architecture

D Discovery Functions
discoverGC + discoverGroups — search by state + sport, validate via public APIs, register in Firestore. Works across all platforms.
A 8 Platform Adapters
GC, GotSport, TGS, Demosphere, SportsAffinity, Pointstreak, LeagueApps, SportsConnect. Unified standings format.
C Collection Engine
collectLeague + collectAll. Daily standings collection via Cloud Functions. Season monitor detects active/inactive leagues.
F Firestore + Sheets
Firestore for league registry + standings. Google Sheets daily sync for dashboard access. 50K reads/day free tier.
M Season Monitor
Automated health checks. Detects stale data, off-season leagues, and newly-added leagues. Auto-onboards new discoveries.
V Deploy VM
deploy-agent VM for GCP operations. Hosts league-standings service, manages Cloud Function deployments, runs ad-hoc tasks.
